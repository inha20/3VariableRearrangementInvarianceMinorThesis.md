## 그림별 상세 명세

### Figure 01 — 표준 4변수 카르노맵
```
내용: Gray Code 배열의 표준 4변수 카르노맵 (4×4 그리드)
행: AB = 00, 01, 11, 10
열: CD = 00, 01, 11, 10
목적: 이론적 배경 — 분석 기반 설명
참고: Paper 2의 그림 1(01.png) 재활용 가능 여부 확인
```

### Figure 02 — 변수 배열 예시
```
내용: AB/CD, AC/BD, AD/BC 세 대표 배열의 카르노맵 나란히 배치
목적: "변수 배열이란 무엇인가?" 한 장으로 설명
형식: 3개 카르노맵 + 제목 레이블
```

### Figure 03 — 변수 재배열 개념
```
내용: AB/CD → AC/BD → AD/BC 화살표 연결
동일 XOR 함수, 배열만 다른 세 카르노맵
목적: 변수 재배열 = 논리는 같고 시각만 바뀜
```

### Figure 16 — 24→3 분류 결과 (핵심 그림)
```
내용: 
  24개 배열
   ↓ (동치 변환)
  대표 배열 A (AB/CD)
  대표 배열 B (AC/BD)
  대표 배열 C (AD/BC)
목적: 연구의 핵심 결론 시각화
```

### Figure 18 — XOR 패턴 변환 (핵심 그림)
```
내용: 동일 XOR 함수를 3가지 대표 배열로 표시
  AB/CD XOR → AC/BD XOR → AD/BC XOR
  각 배열에서 다른 체커보드 방향
목적: 패턴이 변환되지만 구조는 유지됨을 시각적으로 증명
```

### Figure 19 — 변화하는 요소
```
내용: 변수 재배열 전후 비교
  변화: 위치(Position), 방향(Orientation), 대칭축(Symmetry Axis)
  예시: ↘ 대각선 → ↗ 대각선
```

### Figure 20 — 유지되는 구조
```
내용: 동일 함수, 재배열 전후 나란히 배치
  표기: "1의 개수 = 8", "XOR 관계 유지", "반복 구조 유지"
```
