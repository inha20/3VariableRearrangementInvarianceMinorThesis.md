### Figure 01 — 표준 4변수 카르노맵

**파일명:** `figure01_standard_4variable_kmap.png`  
**논문 위치:** 이론적 배경 (제일 첫 그림)  
**목적:** 분석 기반이 되는 표준 카르노맵 구조 설명

**제작 명세:**
```
4×4 그리드 카르노맵

       CD  
AB  | 00 | 01 | 11 | 10 |
 00 |    |    |    |    |
 01 |    |    |    |    |
 11 |    |    |    |    |
 10 |    |    |    |    |

- 변수 레이블: AB (행), CD (열)
- 각 행/열 헤더에 Gray Code 값 표시: 00, 01, 11, 10
- 내부 셀은 빈 상태 (회색 배경 없이, 또는 밝은 회색으로 구별)
- 왼쪽 상단: "AB \ CD" 대각선 구분자
- 캡션: "그림 1. 표준 4변수 카르노맵 (Gray Code 배열). 
  행(AB)과 열(CD) 모두 Gray Code 순서(00, 01, 11, 10)로 배열되며,
  인접한 셀 사이의 Hamming 거리가 1임을 보장한다."
```

**Paper 2 재활용 가능성:** Paper 2의 `01.png`(그림 1)를 참고하거나 직접 재활용 검토
